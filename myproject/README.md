# KCC Olympiad System

A web-based competition management system built using Django.

## Features

- Team login system
- Admin dashboard
- Question approval system
- Leaderboard
- Real-time scoring
- Multiple competition rounds

## Technology Used

- Django
- Python
- HTML/CSS
- SQLite

## Installation

1. Clone the repository

git clone https://github.com/aarlozz/kcc_olympaid.git

2. Navigate to project

cd myproject

3. Create virtual environment

python -m venv myenv

4. Activate environment

myenv\Scripts\activate

5. Install dependencies

pip install -r requirements.txt

6. Run migrations

python manage.py migrate

7. Run server

python manage.py runserver
